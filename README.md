# Homie beamer control for NEC rs232 interface 

Needed Hardware:
* EPS8266 NodeMCU

Configuration:
* use idy.py menuconfig to set parameters (e.g. WiFi Password, MQTT Settings...)

Software:
* vscode with remote containers extension 
* Platform.io
